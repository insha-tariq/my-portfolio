My Skills

  • HTML The building blocks of a webpage. It defines the structure and content of a webpage, like headings, paragraphs, images, and links. Imagine it as the skeleton of a website.
  • CSS Dictates how HTML is presented on a screen. It controls the look and feel of a webpage, like fonts, colors, layouts, and animations. Think of it as the styling for the website's skeleton.
  • JavaScript Adds interactivity to webpages. It allows for dynamic content, user interaction, and complex animations. Imagine it as the muscles and nerves that bring the website to life.
  • ReactA popular JavaScript library for building user interfaces. It allows for creating reusable components and makes building complex web applications easier. Think of it as a pre-built toolkit to help you assemble website features efficiently.
  • NodeJS An environment that allows you to run JavaScript code outside of a web browser. This enables building web servers and applications that run on the server side. Imagine it as a JavaScript engine that can power entire websites, not just what you see in the browser.
